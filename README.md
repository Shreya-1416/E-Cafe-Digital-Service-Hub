# E-Cafe: Digital Service Hub

A full-stack web platform designed to provide digital services like form filling, printing, and ticket booking.

## Features
- Online form filling
- Print order system
- Ticket booking system
- Responsive UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python / Node.js

## Purpose
To simplify access to digital services for users in a centralized platform.
